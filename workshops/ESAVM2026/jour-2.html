Boucles, grilles && images

On commence à créer des systèmes.

Rappel

Avant d'aller plus loin : une variable est une valeur qui peut changer ou être qui peut être partagée.


let x = 0;

function draw() {
  background(220);

  let r = 50;
  rect(150, 150, r, r);

  circle(x, 200, r);
  x = x + 2;  // prends x, ajoute 2, range le résultat dans x
}

Si x était déclaré à l'intérieur de draw(), il serait recréé à zéro à chaque frame. Son scope ; l'endroit où il est déclaré ; détermine où il existe et si il persiste.

Couleurs

On peut définir les couleurs de plusieurs façons :


// couleurs
// fill() pour la couleur de remplissage
fill(255, 0, 0); // rouge (R, G, B)
fill(0, 255, 0, 127); // vert semi-transparent (R, G, B, A)
fill('turquoise'); // une couleur nommée (voir la liste ci-dessous)
fill('#c35272'); // couleur hexadecimale
noFill(); // pas de couleur de remplissage

// stroke() pour la couleur du contour
stroke(0); // contour noir
strokeWeight(3); // épaisseur du contour
noStroke(); // pas de contour

Pour connaître la liste des couleurs nommées disponibles : CSS Color List

Pour stocker une couleur dans une variable :


  let c = color(255, 204, 0); // ou n'importe quelle autre notation

  fill(c);
  background(c);
  stroke(c);

Tableaux

Un tableau permet de stocker plusieurs valeurs dans une seule variable :

let sizes = [30, 60, 40];  // 3 valeurs dans une variable
let formes = ["circle", "rectangle", "line"];

On accède à une valeur par son index (en commençant par 0) :

sizes[0]   // 30 ; premier élément
sizes[1]   // 60 ; deuxième élément
sizes[2]   // 40 ; dernier élément

Le premier index est 0, pas 1 ; un tableau de 3 éléments va de [0] à [2].

La boucle for

5 cercles en 3 lignes de code. Changez 5 en 50 pour avoir 50 cercles ; toujours en 3 lignes.

for (let i = 0; i < 5; i++) {
  let x = i * 80;
  circle(x, 200, 40);
}

On l'a vu, on compte toujours à partir de 0.

À chaque itération, i augmente et la position change.

Structure : for (initialisation; condition; incrément)

On utilise aussi les boucles pour parcourir un tableau :


let posX = [100, 200, 300];
let sizes = [30, 60, 40];
let colors = ["red", "green", "blue"];

for (let i = 0; i < sizes.length; i++) {
  let s = sizes[i];
  let x = posX[i];
  fill(colors[i]);
  circle(x, 200, s);
}

La grille ; boucles imbriquées

i parcourt les colonnes → ; j parcourt les lignes ↓.

À chaque combinaison (i, j) correspond une cellule.

let cols = 10;
let rows = 10;
let cellSize;

function setup() {
  createCanvas(500, 500);

  // une variable peut être calculée à partir d'autres ; ça évite les erreurs et facilite les changements
  cellSize = width / cols;
}

function draw() {
  background(240);

  for (let i = 0; i < cols; i++) {
    for (let j = 0; j < rows; j++) {
      let x = i * cellSize;
      let y = j * cellSize;
      circle(x, y, cellSize);
    }
  }
}

Du dessin procédural au code

Hier, avec un dé sur une grille 3×3 on avait :

  • si pair : /
  • si impair : \

En code, on pourrait faire :


if ((i + j) % 2 === 0) {
  circle(x, y, cellSize);
} else {
  rect(x, y, cellSize, cellSize);
}

C'est la même logique que le dé.

Le temps ; frameCount

frameCount est une variable qui augmente à chaque frame. On peut l'utiliser pour faire évoluer n'importe quel paramètre.


let taille = frameCount * 0.01; // frameCount augmente vite, on le multiplie par un petit nombre pour que ce soit plus lent
circle(x, y, taille);
