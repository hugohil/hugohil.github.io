Aléatoire && interaction

Introduire la variation et le dialogue avec l'utilisateur. Les systèmes commencent à répondre.

random()

random();              // entre 0 et 1
random(100);           // entre 0 et 100
random(20, 80);        // entre 20 et 80

let shapes = ['circle', 'rectangle', 'line'];
let choice = random(shapes);  // élément aléatoire du tableau

Du dé au code

let r = round(random(1, 6));  // valeur entre 1 et 6

if (r == 1)      { /* le dé est tombé sur 1 */ }
else if (r == 2) { /* le dé est tombé sur 2 */ }
else if (r == 3) { /* le dé est tombé sur 3 */ }
else if (r == 4) { /* le dé est tombé sur 4 */ }
else if (r == 5) { /* le dé est tombé sur 5 */ }
else            { /* le dé est tombé sur 5 */ }

Notez qu'on a arrondi le résultat de random(1, 6) pour obtenir un nombre entier. Il existe 3 méthodes pour arrondir : round(), floor() et ceil().

Fixer l'aléatoire ; randomSeed()

Par défaut, random() produit un résultat différent à chaque exécution. Pour obtenir toujours le même output :

function setup() {
  createCanvas(500, 500);
  
  randomSeed(42);  // n'importe quel nombre
  random(); // toujours la même valeur à chaque exécution, tant que la seed est la même
}

C'est utile pour travailler sur un résultat précis, itérer dessus, puis changer la seed pour explorer d'autres variations.

noise() ; le hasard organique

random() produit des valeurs indépendantes, qui peuvent varier de manière abrupte.

noise() simule des valeurs continues : deux appels proches donnent des valeurs proches.

// random() : aucun lien entre valeurs successives
let a = random();   // ex: 0.82
let b = random();   // ex: 0.14  ; sans rapport avec a

// noise() : continuité garantie
let c = noise(0.1);   // ex: 0.54
let d = noise(0.11);  // ex: 0.55  ; proche de c

noise() dans la grille

let n = noise(i * 0.3, j * 0.3);      // valeur entre 0 et 1
let taille = map(n, 0, 1, 5, cellSize);
circle(x, y, taille);

// ajouter une dimension temporelle
let t = frameCount * 0.005;
let n = noise(i * 0.3, j * 0.3, t);  // le motif évolue dans le temps

Changer 0.3 : plus petit → plus fluide, plus grand → plus chaotique.

Interaction ; souris

mouseX et mouseY sont des variables qui contiennent les coordonnées de la souris à chaque frame.

// la taille dépend de la position horizontale de la souris
let taille = map(mouseX, 0, width, 5, cellSize);
circle(x, y, taille);

// l'offset du noise dépend de la souris
let offsetX = map(mouseX, 0, width, 0, 2);
let n = noise(i * 0.3 + offsetX, j * 0.3);

On peut aussi utiliser la souris pour détecter les clics :

function mousePressed() {
  // appelé une seule fois à chaque clic
  cellSize = cellSize + 5;
}

Interaction ; clavier

function keyPressed() {
  // une seule fois, lorsque que la touche est enfoncée
  if (key === 's') {
    saveCanvas('capture', 'png');
  }
  if (key === ' ') {
    cellSize = cellSize + 5;
  }
}

Combiner

let offsetX = map(mouseX, 0, width, 0, 2);
let t = frameCount * 0.005;

for (let i = 0; i < cols; i++) {
  for (let j = 0; j < rows; j++) {
    let x = i * cellSize + cellSize / 2;
    let y = j * cellSize + cellSize / 2;

    let n = noise(i * 0.3 + offsetX, j * 0.3, t);
    let taille = map(n, 0, 1, 4, cellSize * 0.9);

    circle(x, y, taille);
  }
}

Exporter une image

function keyPressed() {
  if (key === 's') saveCanvas('capture', 'png');
}

Pour capturer une frame précise, appeler noLoop() juste avant saveCanvas(). Relancer avec loop().
