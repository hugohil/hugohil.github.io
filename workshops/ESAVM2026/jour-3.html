Mouvements organiques ; pulsation

sin() produit une valeur oscillant entre −1 et 1.

cos() suit le même mouvement que sin() mais décalé d'un quart de cycle


let osc = sin(frameCount * 0.02);
let taille = map(osc, -1, 1, 0, 40);
circle(x, y, taille);

map() ; traduire une plage de valeurs

// map(valeur, min_entrée, max_entrée, min_sortie, max_sortie)
let t = sin(frameCount * 0.02); // entre -1 et 1
let taille = map(t, -1, 1, 0, width); // entre 0 et width

map() traduit une valeur d'une plage vers une autre. Vous allez l'utiliser souvent.

random()

random();              // entre 0 et 1
random(100);           // entre 0 et 100
random(20, 80);        // entre 20 et 80

let shapes = ['circle', 'rectangle', 'line'];
let choice = random(shapes);  // élément aléatoire du tableau

Du dé au code

let r = round(random(1, 6));  // valeur entre 1 et 6

if (r == 1)      { /* le dé est tombé sur 1 */ }
else if (r == 2) { /* le dé est tombé sur 2 */ }
else if (r == 3) { /* le dé est tombé sur 3 */ }
else if (r == 4) { /* le dé est tombé sur 4 */ }
else if (r == 5) { /* le dé est tombé sur 5 */ }
else            { /* le dé est tombé sur 5 */ }

Notez qu'on a arrondi le résultat de random(1, 6) pour obtenir un nombre entier. Il existe 3 méthodes pour arrondir : round(), floor() et ceil().

Fixer l'aléatoire ; randomSeed()

Par défaut, random() produit un résultat différent à chaque exécution. Pour obtenir toujours le même output :

function setup() {
  createCanvas(500, 500);
  
  randomSeed(42);  // n'importe quel nombre
  random(); // toujours la même valeur à chaque exécution, tant que la seed est la même
}

C'est utile pour travailler sur un résultat précis, itérer dessus, puis changer la seed pour explorer d'autres variations.

noise() ; le hasard organique

random() produit des valeurs indépendantes, qui peuvent varier de manière abrupte.

noise() simule des valeurs continues : deux appels proches donnent des valeurs proches.


// random() : changements chaotiques
let a = random();   // ex: 0.82
let b = random();   // ex: 0.14  ; sans rapport avec a

// noise() : plus harmonieux
let c = noise(0.1);   // ex: 0.54
let d = noise(0.11);  // ex: 0.55  ; proche de c

// example avec les couleurs
fill(random(255), random(255), random(255));
circle(100, height / 2, 150);

let t = frameCount * 0.1;
fill(noise(t) * 255, noise(t + 1234) * 255, noise(t + 4567) * 255);
circle(300, height / 2, 150);

Interaction ; souris

mouseX et mouseY sont des variables qui contiennent les coordonnées de la souris à chaque frame.

// la taille dépend de la position horizontale de la souris
let taille = map(mouseX, 0, width, 5, cellSize);
circle(x, y, taille);

// l'offset du noise dépend de la souris
let offsetX = map(mouseX, 0, width, 0, 2);
let n = noise(i * 0.3 + offsetX, j * 0.3);

On peut aussi utiliser la souris pour détecter les clics :

function mousePressed() {
  // appelé une seule fois à chaque clic
  cellSize = cellSize + 5;
}

Combiner

let offsetX = map(mouseX, 0, width, 0, 2);
let t = frameCount * 0.005;

for (let i = 0; i < cols; i++) {
  for (let j = 0; j < rows; j++) {
    let x = i * cellSize + cellSize / 2;
    let y = j * cellSize + cellSize / 2;

    let n = noise(i * 0.3 + offsetX, j * 0.3, t);
    let taille = map(n, 0, 1, 4, cellSize * 0.9);

    circle(x, y, taille);
  }
}

Transformations ; push() && pop()

On permet de déplacer (translation), tourner (rotation) et redimensionner (scale) le système de coordonnées entier plutôt que de calculer chaque position à la main.

translate(x, y);   // déplace l'origine
rotate(angle);     // tourne autour de l'origine (en radians)
scale(facteur);    // redimensionne

Ces transformations s'accumulent. push() sauvegarde l'état actuel, pop() et le restaure ; ça permet d'isoler les transformations :


rectMode(CENTER);

// on démarre les transformations
push();
translate(width / 2, height / 2); // on déplace l'origine au centre
rotate(frameCount * 0.02); // on tourne

rect(0, 0, 50, 50);
circle(100, 0, 50, 50);
pop();
// on arrête les transformations

rect(0, 0, 50, 50);
circle(100, 0, 50, 50);

Sans push()/pop(), chaque rotate() dans une boucle s'additionne au précédent. Avec, on peut isoler les transformations.

Texte && typographie

On peut afficher et manipuler du texte comme n'importe quelle forme.

textSize(32);
textAlign(CENTER, CENTER);  // alignement horizontal et vertical
fill(0);
text("hello", x, y);        // affiche le texte à la position (x, y)

On peut charger une police externe depuis Google Fonts — dans le HTML, avant le sketch :

<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=IBM+Plex+Mono&display=swap">
function setup() {
  createCanvas(500, 500);
  textFont('IBM Plex Mono');
}

Comme pour les formes, toutes les propriétés peuvent être pilotées par des variables :


let n = noise(frameCount * 0.025);
let ts = map(n, 0, 1, 12, 48);
textSize(ts);

textAlign(CENTER, CENTER);
text("hello", width / 2, height / 2);
